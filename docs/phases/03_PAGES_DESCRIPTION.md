# 📄 وصف كل صفحة في المنصة

## الصفحات العامة (بدون تسجيل دخول)

### 1. Landing Page - صفحة الهبوط (/)
**الغرض:** تسويق المنصة وتحويل الزوار لمشتركين.

**الأقسام:**
- **Navbar:** لوغو + روابط (الميزات، الأسعار، API، تواصل) + زر دخول + زر تسجيل
- **Hero:** عنوان رئيسي + وصف + زر "ابدأ مجاناً" + صورة dashboard
- **Stats:** إحصائيات مقنعة (عملاء، رسائل، دول)
- **Features 6:** AI ردود، حملات، رد تلقائي، API، أزرار تفاعلية، توثيق للمطورين
- **How It Works:** 3 خطوات: ربط جهاز → بناء بوت → أرسل وتتبع
- **Pricing:** بطاقات الخطط الثلاث مع مقارنة كاملة
- **Testimonials:** آراء عملاء حقيقية
- **FAQ:** أسئلة شائعة
- **CTA:** دعوة أخيرة للتسجيل
- **Footer:** روابط + سوشيال + حقوق + سياسة الخصوصية

**الوظائف التقنية:**
- تبديل اللغة عربي/إنجليزي
- Smooth scroll بين الأقسام
- Animation عند الـ scroll
- زر واتساب للتواصل المباشر
- Cookie consent

---

### 2. Register - التسجيل (/register)
**الحقول:**
- الاسم الكامل (مطلوب)
- البريد الإلكتروني (مطلوب، unique)
- رقم الهاتف مع كود الدولة (مطلوب)
- كلمة المرور (min 8 أحرف)
- تأكيد كلمة المرور
- كود الإحالة (اختياري - يُملأ تلقائياً من ?ref=)
- موافقة على الشروط والأحكام

**بعد التسجيل:**
- إرسال OTP للتحقق من رقم الهاتف
- توليد API Key تلقائياً
- توليد كود إحالة فريد
- بدء فترة تجربة مجانية تلقائياً (حسب الخطة)
- توجيه للـ Dashboard

---

### 3. Login - تسجيل الدخول (/login)
**خيارات:**
- بريد إلكتروني + كلمة مرور
- رقم هاتف + OTP
- "نسيت كلمة المرور"

### 5. Devices - الأجهزة (/devices)
**عنوان الصفحة:** "حساب WhatsApp"

**الجدول:**
| الرقم | رابط Webhook | مرسلة | الحالة | خيارات | خيارات إضافية |

**الحالات:**
- 🟢 متصل
- 🔴 منقطع
- ⚠️ منتهي الصلاحية
- 🚫 محظور

**نموذج إضافة جهاز:**
- رقم الهاتف (مع كود الدولة بدون +)
- اسم الجهاز
- رابط Webhook (اختياري)

**آلية الربط:**
1. إدخال الرقم → Backend ينشئ جلسة Baileys
2. توليد QR Code → عرضه عبر Socket.IO
3. المستخدم يمسح QR من واتساب
4. الجهاز يصبح "متصل"

**خيارات لكل جهاز:**
- إعدادات الجهاز (تعديل Webhook، اسم، AI)
- تفعيل/تعطيل الذكاء الاصطناعي
- System Prompt للـ AI
- فصل الجهاز
- حذف الجهاز

**الحد حسب الخطة:** Basic=1, Pro=3, Business=10

### 8. Campaigns - الحملات (/campaigns)
**نموذج إنشاء حملة:**
- اسم الحملة
- اختيار الجهاز
- نوع المستلمين: قائمة جهات اتصال / أرقام يدوية / رفع ملف
- نوع الرسالة: نص، صورة، فيديو، مستند، موقع، منتج، قائمة، أزرار، poll، sticker
- محتوى الرسالة (مع دعم {{اسم_المتغير}})
- التأخير بين الرسائل: عشوائي بين X-Y ثانية
- جدولة: فوري أو وقت محدد

**حالات الحملة:** Draft → Scheduled → Running → Paused → Completed/Failed

**الإحصائيات:** % النجاح، مرسلة، فاشلة، منتظرة، الحملات

### 10. Auto Reply - الرد التلقائي (/autoreply)
**أنواع التشغيل:**
- `keyword` - كلمة مفتاحية محددة
- `contains` - تحتوي على
- `starts_with` - تبدأ بـ
- `ends_with` - تنتهي بـ
- `first_message` - أول رسالة من مستخدم جديد
- `all` - كل الرسائل
- `outside_hours` - خارج ساعات العمل

**أنواع الردود:** نص، صورة، فيديو، قائمة، أزرار، تدفق محادثة، قالب

**إعدادات إضافية:**
- ساعات العمل (من - إلى)
- أيام العمل (الأحد - السبت)
- الأولوية (لو أكثر من قاعدة تنطبق)
- عداد الاستخدام

### 11. Chat Flow Builder (/chatflow)
**الواجهة:** كانفاس Drag & Drop مع React Flow

**أنواع العقد (Nodes):**

| النوع | الوظيفة |
|-------|---------|
| Start | نقطة البداية (كلمة مفتاحية) |
| Message | إرسال رسالة (أي نوع) |
| Question | سؤال مع خيارات |
| Condition | شرط if/else |
| AI Node | إرسال للذكاء الاصطناعي |
| Delay | انتظار X ثانية/دقيقة/ساعة |
| Set Variable | تعيين متغير |
| HTTP Request | طلب لـ API خارجي |
| End | نهاية التدفق |

**الإعدادات لكل عقدة:**
- المحتوى (نص، صورة، قائمة...)
- الشروط للانتقال للعقدة التالية
- Timeout قبل الانتقال التلقائي

**حفظ ومشاركة:** JSON قابل للتصدير/الاستيراد

### 20. Referral Center - مركز الإحالات (/referrals)
**الإحصائيات:**
- الأرباح المتاحة للسحب (SAR) - أخضر
- الأرباح المحجوزة (SAR) - برتقالي
- إجمالي الأرباح (SAR) - بنفسجي
- إجمالي الإحالات الناجحة

**الأدوات:**
- كود الإحالة الفريد (نسخ بنقرة)
- رابط الإحالة (نسخ + مشاركة)
- أزرار مشاركة (واتساب، تويتر، تيليغرام)

**السحب:**
- حقل المبلغ
- الحد الأدنى: 25 SAR
- طريقة: تحويل بنكي
- بيانات البنك (IBAN، اسم صاحب الحساب، البنك)

**الجداول:**
1. المستخدمون المُحالون (الاسم، البريد، تاريخ الانضمام)
2. جدول العمولات (المستخدم، المبلغ، الحالة، تاريخ التوفر)
3. سجل السحوبات (المبلغ، الطريقة، الحالة، التاريخ)
